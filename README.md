# Nettools

A simple, small, alpine-based Docker image with some handy networking tools
installed:

- bpfc
- curl
- drill
- ethtool
- iperf3
- iproute2
- iptables
- mtr
- netsniff-ng
- nftables
- openssl
- socat
- tcpdump
- telnet
- trafgen
- tshark / dumpcap / mergecap

Please see the Dockerfile for a complete list of tools.
